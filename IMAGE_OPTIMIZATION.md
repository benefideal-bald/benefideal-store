# Оптимизация изображений для мобильных устройств

## Проблема
Текущие изображения товаров очень большие:
- ChatGPT: **3.4 MB**
- Adobe: **3.4 MB**
- CapCut: **3.7 MB**

Это вызывает медленную загрузку на слабых устройствах и мобильных сетях.

## Решения

### 1. Сжатие PNG (быстрое решение)
Используйте инструменты для сжатия PNG без потери качества:
- **TinyPNG** (https://tinypng.com/) - онлайн сжатие
- **ImageOptim** (Mac) - приложение для сжатия
- **Squoosh** (https://squoosh.app/) - онлайн оптимизатор

**Цель:** Уменьшить размер до **200-500 KB** на изображение

### 2. Конвертация в WebP (лучшее решение)
WebP формат дает на 25-35% меньший размер при том же качестве:
- Используйте **Squoosh** или **ImageMagick**
- Создайте `.webp` версии изображений
- Обновите HTML для использования WebP с fallback на PNG

### 3. Адаптивные изображения
Используйте разные размеры для мобильных и десктопа:
```html
<picture>
  <source srcset="images/ChatGPT-mobile.webp" media="(max-width: 768px)" type="image/webp">
  <source srcset="images/ChatGPT-mobile.png" media="(max-width: 768px)" type="image/png">
  <source srcset="images/ChatGPT-desktop.webp" type="image/webp">
  <img src="images/ChatGPT dfghjk.png" alt="Chat-GPT Plus" loading="eager" fetchpriority="high">
</picture>
```

### 4. Рекомендуемые размеры
- **Мобильные:** 600-800px ширина, WebP, ~150-300 KB
- **Десктоп:** 1200-1600px ширина, WebP, ~300-500 KB

## Инструкция по оптимизации

1. Откройте изображение в **Squoosh** (https://squoosh.app/)
2. Выберите формат **WebP**
3. Установите качество **80-85%**
4. Для мобильных версий уменьшите ширину до **800px**
5. Сохраните оптимизированные версии
6. Замените файлы в папке `images/`

## Ожидаемый результат
- Загрузка на мобильных: **с 5-10 секунд до 1-2 секунд**
- Экономия трафика: **~90%** (с 3.5MB до ~300KB)
- Улучшение производительности на слабых устройствах
