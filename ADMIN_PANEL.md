# Админ-панель - Инструкция

## Доступ к админ-панели

1. Откройте страницу: `https://benefideal.ru/admin.html`
2. Введите пароль (по умолчанию: `admin123`)
3. Или используйте пароль через URL: `https://benefideal.ru/admin.html?password=ВАШ_ПАРОЛЬ`

## Настройка пароля

Для безопасности измените пароль по умолчанию:

1. На Render перейдите в настройки вашего сервиса
2. Добавьте переменную окружения:
   - **Key:** `ADMIN_PASSWORD`
   - **Value:** ваш надежный пароль
3. Сохраните и перезапустите сервис

## Что отображается в админ-панели

### Статистика
- **Всего заказов** - общее количество заказов
- **Общая сумма** - сумма всех заказов в рублях
- **Активных** - количество активных подписок
- **Сегодня** - количество заказов за сегодня

### Таблица заказов
Для каждого заказа отображается:
- **ID** - уникальный идентификатор заказа
- **Имя** - имя клиента
- **Email** - электронная почта клиента
- **Товар** - название купленного товара
- **Срок** - срок подписки (месяцы)
- **Сумма** - сумма платежа в рублях
- **Дата** - дата покупки
- **Время** - время покупки
- **Статус** - активен/неактивен

### Фильтры
- **Поиск по имени/email** - поиск по имени или email клиента
- **Товар** - фильтр по конкретному товару
- **Дата от/до** - фильтр по диапазону дат

## Обновление данных

Нажмите кнопку **"Обновить"** в правом верхнем углу, чтобы загрузить свежие данные из базы.

## Безопасность

⚠️ **Важно:** 
- Не используйте пароль по умолчанию в продакшене
- Установите надежный пароль через переменную окружения `ADMIN_PASSWORD`
- Не делитесь паролем с посторонними
- Админ-панель доступна только по паролю
